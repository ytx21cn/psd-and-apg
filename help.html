访问键帮助

可用操作

  • B: 返回主页
  • C: 关门
  • H: 显示此帮助页
  • O: 开门

在不同浏览器中使用访问键

Internet Explorer

  • Windows: Alt + 访问键

Google Chrome

  • Windows/Linux: Alt + 访问键
  • Mac: Control + Alt + 访问键

Mozilla Firefox

  • Windows/Linux: Alt + Shift + 访问键
  • Mac: Control + Alt + 访问键

Apple Safari

  • Windows: Alt + 访问键
  • Mac: Control + Alt + 访问键

Opera

  • Opera 15或更新：Alt + 访问键
  • Opera 12.1或更旧： Shift + Esc + 访问键
